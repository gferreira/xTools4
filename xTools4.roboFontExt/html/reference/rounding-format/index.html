Rounding format

  • «
  • »
  1. Index
  2. Reference
  3. rounding format

A data format to store points which are modified to produce rounded sources.

Python example

{
    'D' : {
        'corners' : [0, 1],
    },
    'E' : {
        'caps' : [(2, 3), (10, 11), (6, 7)],
        'corners' : [0, 1],
    },
    'I' : {
        'caps' : [(1, 2), (3, 0)],
    },
    'T' : {
        'caps' : [(4, 5), (6, 7), (3, 0)],
    },
}
