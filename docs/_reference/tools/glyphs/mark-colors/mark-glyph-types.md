Apply mark colors to glyphs based on type: contour, components, contours & components, empty.
{: .lead }


<div class='row'>
<div class='col-4' markdown='1'>
![]({{ site.url }}/images/glyphs/MarkGlyphTypes.png){: .img-fluid }
</div>
<div class='col-8' markdown='1'>
contours
: Choose a color for glyphs which consist entirely of contours.

components
: Choose a color for glyphs which consist entirely of components.

contours & components
: Choose a color for glyphs which consist of both contours and components.

empty
: Choose a color for empty glyphs (no contours or components).

apply
: Apply the chosen mark colors to the selected glyphs.
