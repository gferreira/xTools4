Overlay glyphs from other UFOs in the background of the current glyph.
{: .lead }

<div class='row'>

<div class='col' markdown='1'>
![]({{ site.url }}/images/glyph/overlay-ufos.png){: .img-fluid}
</div>

<div class='col' markdown='1'>
UFOs
: A list of UFO fonts which can be displayed in the background.  
  Fonts which are already open are included automatically.  
  Other UFOs can be added by dragging files from Finder into the list.  
  Select one or more fonts to display in the background.

color
: Choose a color for the overlaid UFOs.

display options
: Choose which glyph attributes to display.

alignment
: Choose between aligning glyphs to the left, center or right side.
</div>

</div>
